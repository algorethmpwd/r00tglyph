Mission Control
UNION-based SQL Injection

Welcome to ProductCatalog. Test for SQL injection vulnerabilities.

Objective

Extract data using UNION SELECT

UNION allows combining results from multiple SELECT statements. First, determine the number of columns in the original query by using: ' ORDER BY 1--, ' ORDER BY 2--, etc. until you get an error.

Once you know the column count, use UNION to extract data: ' UNION SELECT username, password FROM users--. Match the number of columns in your UNION with the original query.

Step-by-step:
  1. Find column count: ' ORDER BY 3-- (adjust until no error)
  2. Use UNION: ' UNION SELECT 1,username,password FROM users--
  3. View the results to see extracted user data
Why it works: UNION combines your malicious SELECT with the original query, displaying sensitive data.
