Objective

Exploit NoSQL injection in MongoDB

NoSQL databases like MongoDB use JSON queries. Inject operators: {"username": {"$ne": null}} to bypass authentication.

Use $ne (not equal), $gt (greater than), $regex for injection. Example: username[$ne]=null&password[$ne]=null

In login form, use: username[$ne]=null&password[$ne]=null to bypass authentication by making the query always match.
