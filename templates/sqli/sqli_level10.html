Objective

Bypass Web Application Firewall

WAFs block common SQLi patterns. Test what's blocked: UNION, SELECT, OR, etc. Then find bypass techniques.

Use: Case variations (UnIoN), comments (UN/**/ION), encoding (%55NION), or alternative syntax (|| instead of OR).

Bypass examples:
  • ' /*!50000UNION*/ /*!50000SELECT*/ 1,2,3--
  • ' %55NION %53ELECT 1,2,3--
  • ' UNION/**/SELECT/**/1,2,3--
Why it works: WAF rules miss obfuscated variations.
