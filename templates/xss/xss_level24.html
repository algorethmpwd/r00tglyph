Objective

Inject HTML elements with IDs that clobber the global config variable.

HTML elements with an id attribute are automatically added to the global window scope as variables.

If the application uses a global variable like config without initializing it with `let` or `const`, you can overwite it by injecting an element with `id="config"`.
