Level 27: Client-Side Template Injection (CSTI)

Welcome to Reactive Dashboard. Powered by Alpine.js.

Objective

Inject Alpine.js directives to execute arbitrary JavaScript.

Frameworks like Alpine.js, Vue, and Angular scan the DOM for special attributes (directives) to execute logic.

If user input is injected into an HTML attribute that the framework scans, you can inject directives like x-on:click or x-init.

Payload: <div x-data x-init="alert(1)"></div> or breaking out of an attribute to add x-on:click="alert(1)".
