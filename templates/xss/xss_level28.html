Welcome to Module Loader. Managing ES Modules dependencies.

Objective

Hijack module imports using Import Maps.

Import maps allow web pages to control the behavior of JavaScript imports, mapping bare module specifiers to URLs.

If you can inject a <script type="importmap"> before the application modules load, you can redirect safe imports to malicious scripts.

Payload: <script type="importmap">{"imports": {"/js/utils.js": "data:text/javascript,alert(1)"}}</script>
