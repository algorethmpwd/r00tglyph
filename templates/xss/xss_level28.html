Welcome to Module Loader. Hijack module imports using Import Maps to execute arbitrary code.

Objective

Inject a malicious Import Map to redirect critical module requests to your payload.

Import maps allow web pages to control the behavior of JavaScript imports, mapping bare module specifiers to URLs.

If you can inject a <script type="importmap"> before the application modules load, you can redirect safe imports to malicious scripts.

Payload: <script type="importmap">{"imports": {"/static/js/dummy_utils.js": "data:text/javascript,alert(1)"}}</script>
