Objective

Simulate a malicious HMR update message to execute code.

Hot Module Replacement (HMR) allows modules to be exchanged in a running application without a full reload using WebSockets.

If an attacker can inject messages into the HMR WebSocket channel, they can push a malicious "update" that the client executes.
