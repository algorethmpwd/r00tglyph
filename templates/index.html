R00tGlyph - Web Security Training

About R00tGlyph

R00tGlyph is an advanced web security training platform designed to simulate real-world vulnerabilities in a safe, controlled environment. Perfect for security researchers, penetration testers, and developers looking to improve their security skills.

Each challenge comes with a unique flag that changes for every user, ensuring a personalized experience and preventing flag sharing.

Current Vulnerabilities

  • Cross-Site Scripting (XSS) 23 Levels
  • SQL Injection (SQLi) 7 Levels
  • Command Injection Coming Soon

How to Use R00tGlyph

Choose a Challenge

Browse the available challenges and select one that matches your skill level. Start with beginner challenges if you're new.

Exploit the Vulnerability

Use your favorite tools and techniques to find and exploit the vulnerability in the challenge.

Capture the Flag

Successfully exploiting the vulnerability will reveal a unique flag that you can submit to earn points.

Learn & Improve

Each challenge includes a detailed solution guide explaining the vulnerability and how to prevent it.

Important: This platform is for educational purposes only. The skills learned here should only be applied to systems you have permission to test.
