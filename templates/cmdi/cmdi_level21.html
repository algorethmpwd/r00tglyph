Mission Control
Command Injection in Cloud Functions

Welcome to CloudDeploy. Deploy simple Python cloud functions.

Objective

Inject a command into the function code to execute system commands (`os.system` or `subprocess`).

The deployment process scans for restricted functions but might miss some if obfuscated or if the check is simple pattern matching.

Simply use the banned functions: import os; os.system('id'). The scanner is looking for these to flag them, but in this challenge, finding them *is* the goal (simulating a successful exploit detection or actual execution depending on the backend logic).
